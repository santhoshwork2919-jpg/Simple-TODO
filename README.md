# ✅ Simple TODO

A simple, single-file To-Do list app built with plain HTML, CSS, and JavaScript — no frameworks, no dependencies.

## ✨ Features

- Add new tasks to your list
- Input validation (won't let you add an empty task)
- Delete tasks with a single click
- Clean, minimal, single-file implementation (`TODO.html`)

## 🛠️ Tech Stack

- **HTML** – structure
- **CSS** – styling
- **Vanilla JavaScript** – DOM manipulation and app logic

## ⚙️ How It Works

- Typing a task and clicking "Add" creates a new `<li>` element and appends it to the task list.
- If the input is empty, an alert prompts the user to enter a task before adding.
- Each task comes with a **Delete** button that removes it from the list using `button.parentElement.remove()`.

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/santhoshwork2919-jpg/Simple-TODO.git
   ```
2. Open `TODO.html` directly in your browser — no build steps or installs needed.
